# \## Project Description

# 

# This project implements an interactive physics-based game where players control a tilting plane to collect coins with a rolling ball.

# 

# \### Gameplay

# 

# The game is played by moving a \*\*block\*\* to spawn \*\*coins\*\* across the game environment. Once the coins are placed, the player tilts the game plane to guide the \*\*ball\*\* and collect the coins.

# 

# \### How to Play

# 

# \- Use controls to move the spawning block and place coins in the environment.

# \- Tilt the game plane (using mouse, keyboard, or device tilt depending on platform) to roll the ball.

# \- Collect all coins by maneuvering the ball through the generated environment.
